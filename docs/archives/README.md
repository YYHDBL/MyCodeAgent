# Historical documentation archive

`legacy-harness/` preserves superseded design notes, portfolio material,
roadmaps, protocol drafts, and deterministic-demo snapshots. They are useful
for migration archaeology, but they are not current product documentation and
must not be used as an API or behavior contract.

Current documentation is intentionally limited to:

- [the README](../../README.md) for installation and use;
- [the harness architecture](../HARNESS.md) for current boundaries; and
- the dated [lean-runtime plan](../plans/2026-07-12-lean-runtime/) for the
  migration evidence and release report.
