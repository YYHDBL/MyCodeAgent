我的核心能力

基于 ReAct 范式，为学习 Agent 系统而生的实验框架

统一工具协议

所有工具遵循标准响应格式：status/data/text/stats/context/error，让每次调用都可追溯、可解析、可测试。

Message List 上下文工程

分层注入 L1(系统) / L2(规则) / L3(历史)，支持历史压缩、工具输出截断、会话持久化。

Skills / Task 子代理

内置技能系统与可委托子代理，支持通用/探索/规划/摘要四种模式，实现复杂任务的分而治之。

Trace 可观测性

JSONL + Markdown 双轨日志记录，自动脱敏敏感信息，完整追踪每次决策过程。

MCP 扩展支持

通过 Model Context Protocol 接入外部工具生态，无限扩展能力边界。

轻量熔断机制

连续失败的工具自动临时禁用，避免无效调用浪费 Token，提升稳定性。

快速开始

三步即可启动你的代码助手

Terminal 
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动交互式 CLI
python scripts/chat_test_agent.py

# 3. 或指定自定义模型
python scripts/chat_test_agent.py \
  --provider zhipu \
  --model GLM-4.7 \
  --api-key YOUR_KEY \
  --base-url https://open.bigmodel.cn/api/coding/paas/v4
