Project Architecture

Layered Architecture

The project follows a clean, layered architecture for AI agents built on ReAct (Reasoning + Acting) principles:

  • Core Layer: Foundational components including agent base classes, LLM interfaces, and configuration management
  • Agents Layer: Implementations of various AI agents with specialized capabilities
  • Tools Layer: Tool system with built-in utilities for file operations, code search, and system interactions
  • UI Layer: Modern React interface for seamless user interaction with AI agents

All components communicate through a unified response protocol ensuring consistency and reliability across the agent ecosystem.
